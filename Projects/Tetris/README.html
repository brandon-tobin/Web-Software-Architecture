README

Brandon Tobin and Fumiko Anne Aoki

The project is stored on Brandon's GitHub Repository.

This project is a tetris game that uses Javascript and the PIXI library. To start a game, press the "Start Game" button. Press the down, right, and left arrows to move the piece around or the up arrow to rotate it. Once the game has ended, type your name into the text input area and press "Submit Score" to submit it to the database and see the top five scores. For our additional feature, we added background music to the gameplay. The music has a delayed start.

For css styling, we referenced www.w3schools.com.
