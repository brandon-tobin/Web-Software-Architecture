This project builds on Assignment 2 by implementing a MySQL database with the MVC architecture to build the website. The controller intercepts the requests into the Grad_Progress directory and uses the model to build an object to display to the view. All of the data in the Grad_Progress system is now being pulled from the MySQL database.

I chose to represent the roster, list of all completed forms for each student, and part of the form using a HTML table styled with CSS. Each webpage has a consistent header and horizontal navigation bar for easy navigation. On the backend, there is a MVC architecture with a MySQL database for holding information.

Now that we have a database, I removed all of the hard coded information and now query it into the object from the database. I receive conditions for my queries based on the parameters sent via the url.

For testing, I clicked on every link to make sure that it went to the right webpage. I also tested the website on multiple browsers. I then inspected the code through the feature in PhpStorm. I also made sure all of my calculations are accurate and I tested my SQL queries using MySQL Workbench.

This project builds on Assignment 1 by adding a MVC architecture to build the website. The controller intercepts the requests into the Grad_Progress directory and uses the model to build an object to display to the view.

I chose to represent the roster, list of all completed forms for each student, and part of the form using a HTML table styled with CSS. Each webpage has a consistent header and horizontal navigation bar for easy navigation. On the backend, there is a MVC architecture.

Since we do not have a database yet, I hard coded all of the information into each class object. I use if statements to determine which object should be created based off of the id passed in the url.

For testing, I clicked on every link to make sure that it went to the right webpage. I also tested the website on multiple browsers. I then inspected the code through the feature in PhpStorm.

I did not reference any external pages for this assignment.
