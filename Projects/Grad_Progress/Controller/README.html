For this assignment, I added Highcharts to the DGS Overview page found here to make the page easier to read and display more data.

I believe my work should receive a superior grade because:

  1. I have three charts that display different information for the DGS:
    1. The first chart displays the GPA distribution of the students in a column chart.
    2. The second chart displays how many students each advisor has in a line chart.
    3. The third chart displays how many activities each student has completed in a pie chart.
  2. All of the charts use AJAX to receive the information to populate the chart. When a user selects a different chart, an AJAX call is made to a PHP function that queries the database.
  3. The use of AJAX allows the DGS to be able to switch between charts without having to reload the page every time the DGS switches charts.
  4. The use of the charts allows the DGS to be able to view information about the students and advisors without having to pull up individual records or having to compute the information themselves.
  5. The way I incorporated AJAX with the charts makes it very easy and efficient to add more charts to increase functionality.
