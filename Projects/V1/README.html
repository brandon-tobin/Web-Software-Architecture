This is the starting project for the Grad Progress project. There is a roster of all students, each student has a list of all of their completed forms, and the completed forms for each student. As of right now, all of the information is hard coded, but will be changed in the future. The websie only shows the pages essential for the front end of the Grad Progress project.

I chose to represent the roster, list of all completed forms for each student, and part of the form using a HTML table styled with CSS. Each webpage has a consistent header and horizontal navigation bar for easy navigation.

Once you get to a student webpage, a back button will appear for easy navigation back to the student list and back to the student's form list once viewing their form.

For testing, I clicked on every link to make sure that it went to the right webpage. I also tested the website on multiple browsers.
